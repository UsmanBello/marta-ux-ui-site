# UX/UI Portfolio Website

A modern, Apple-inspired portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Apple-style Design**: Glassmorphism effects, smooth animations, and modern aesthetics
- **Responsive Layout**: Works beautifully on all devices
- **Multiple Pages**: Home, Experience, Portfolio, and Contact pages
- **Smooth Animations**: Powered by Framer Motion
- **Modern Stack**: Next.js 14 with App Router, TypeScript, and Tailwind CSS

## Getting Started

### Install Dependencies

```bash
npm install
# or
yarn install
```

### Run Development Server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Pages

- **Home** (`/`): Hero section, about me, and featured work preview
- **Experience** (`/experience`): Work history timeline and skills
- **Portfolio** (`/portfolio`): Project showcase with filtering
- **Contact** (`/contact`): Contact form and information

## Design Features

- Glassmorphism effects with backdrop blur
- Gradient text and backgrounds
- Smooth page transitions
- Hover animations
- Apple-inspired color scheme
- Modern typography

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
